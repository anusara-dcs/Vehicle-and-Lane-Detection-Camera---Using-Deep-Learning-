# Vehicle-and-Lane-Detection-Camera---Using-Deep-Learning-
A deep learning–based traffic monitoring system that detects vehicles and road lanes from video using YOLO models. The system includes a GUI for video annotation, model training, and real-time processing, supporting intelligent transportation research and lane usage analysis.

Due to GitHub file size limitations, the dataset and trained model files used in this project are not included in this repository.
You can download the required files from the following Google Drive link:
https://drive.google.com/drive/folders/1wTTZdXuLPNX_ep6eQ7bQONtHsbWEzCOR?usp=drive_link 

After downloading, place the files in the appropriate project directories before running the training or detection scripts.
